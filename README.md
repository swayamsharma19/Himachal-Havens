# 🏔️ Himachal Havens

> A full stack travel & stay booking web app for exploring the most beautiful havens across Himachal Pradesh, India.

- 🖥️ **Frontend:** [himachalhavens1.netlify.app](https://himachalhavens1.netlify.app)
- ⚙️ **Backend API:** [himachal-havens.onrender.com](https://himachal-havens.onrender.com)

---

## ✨ Features

- 🏠 Browse stunning stays across Himachal Pradesh — cottages, villas, homestays, camping sites & more
- 🔍 Filter destinations by type
- 📋 Add your own property listing with images, title, price & location
- 📩 Contact form to send inquiries directly to the backend
- 📱 Fully responsive design
- ⚡ Fast and lightweight UI built with React + Vite

## 📁 Project Structure

```
Himachal-Havens/
├── 📁 backend/
│   ├── server.js          # Express server with API routes
│   └── package.json       # Backend dependencies
│
└── 📁 Himachal-Havens/    # React frontend
    ├── src/
    │   ├── Component/
    │   │   ├── Home/       # Main listings page
    │   │   ├── Card/       # Property card component
    │   │   ├── Nav/        # Navigation bar
    │   │   ├── Listing/    # Add new listing form
    │   │   ├── Contactus/  # Contact form
    │   │   ├── Footer/     # Footer
    │   │   ├── Login/      # Login page
    │   │   ├── Signup/     # Signup page
    │   │   └── ...         # Category filter pages
    │   ├── Context/
    │   │   └── Usercontext.jsx  # Global state management
    │   └── assets/         # Property images
    └── package.json
```
